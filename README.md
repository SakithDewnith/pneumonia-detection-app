Chest X-Ray Pneumonia Detection using Deep Learning
Project Overview

This project develops an automated pneumonia detection system using Convolutional Neural Networks (CNNs) to classify chest X-ray images as either Normal or Pneumonia.

The system applies image preprocessing techniques including CLAHE, normalization, and data augmentation before training a CNN model. The trained model assists clinicians by providing fast, reliable pneumonia screening from chest X-ray images.

The project also includes a Streamlit web application that allows users to upload an X-ray image and receive an instant prediction.
